21 สัปดาห์ที่ผ่านมาคุณต่อยอด library-system (โจทย์ที่ให้มา) — Capstone คือการ ออกแบบและสร้างโปรเจกต์เองตั้งแต่ศูนย์ · Week 22 โฟกัสที่ทักษะ senior ที่สุด: ออกแบบระบบก่อนเขียน (design doc, architecture, threat model) แล้ว scaffold + สร้าง vertical slice แรก · จบ 2 สัปดาห์ (22-23) จะได้โปรเจกต์ production-grade ที่เล่า interview ได้ 30 นาที

  • เลือก Capstone (SecureShare หรือ VulnTracker — ทั้งคู่ใช้จุดแข็ง security)
  • เขียน Design Document — requirements, architecture, data model, API, threat model
  • MVP scoping — เลือกสิ่งที่สำคัญสุดทำก่อน (ไม่ทำทุกอย่าง)

  • สร้าง vertical slice แรก (1 feature ครบ frontend→backend→DB)
ทำไมสัปดาห์นี้คือ peak ของคอร์ส: "ออกแบบระบบเองได้" คือนิยามของ senior — ไม่ใช่แค่ทำตามโจทย์ · design doc + threat model เป็นสิ่งที่ interview senior/staff ขอ และเป็น artifact ที่ทำให้ portfolio โดดเด่น

เลือก Capstone

🔐 SecureShare

ระบบแชร์ไฟล์เข้ารหัส — upload ไฟล์ → ได้ลิงก์ที่หมดอายุ + เข้ารหัส + audit log ว่าใครเปิดเมื่อไหร่

ใช้: encryption, expiring link, access control, audit · เด่น: crypto + security

🐛 VulnTracker

เครื่องมือติดตามผล pentest สำหรับทีม — บันทึก finding, severity, status, มอบหมาย, dashboard, report export

ใช้: CRUD, role, workflow, dashboard, export · เด่น: ตรงงาน security คุณ + เล่า interview ดี

ทั้งคู่ใช้จุดแข็ง security ของคุณ · VulnTracker เล่า interview ได้ดี (คุณเข้าใจ pain point จากประสบการณ์จริง) · SecureShare เด่นด้าน crypto · เลือกที่สนุกกว่า

1. Design ก่อนโค้ด — แต่ไม่ over-design

❌ เขียนเลยไม่คิด / หรือ design 2 สัปดาห์ไม่เขียน

// เขียนเลย → architecture พังกลางทาง
// หรือ over-design ทุก edge case → ไม่เริ่มสักที

✅ design พอเห็นภาพ → เริ่ม → ปรับ

// design doc 1-2 หน้า (พอเห็นโครง)
// → vertical slice → เรียนรู้ → ปรับ design
// iterative ไม่ใช่ waterfall

2. Vertical slice — ทำ 1 feature ครบ ดีกว่าทุก feature ครึ่ง ๆ

สร้าง 1 feature ครบ stack (UI→API→DB→test→deploy) ก่อน แล้วค่อยเพิ่ม feature · เห็นทั้งระบบทำงาน early · ดีกว่าทำ backend หมดแล้วค่อยเริ่ม frontend

กฎทอง: design พอเห็นภาพแล้วเริ่ม (ไม่ over-design) · MVP ก่อน (ตัด nice-to-have) · vertical slice (1 feature ครบ) · apply ทุกอย่างที่เรียน (TS/test/Docker/CI/security) ตั้งแต่ต้น · threat model ตั้งแต่ design
