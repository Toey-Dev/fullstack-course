🧹 Clean Code ประจำ Week 0 — เขียนแบบมืออาชีพตั้งแต่บรรทัดแรก

รูปแบบที่ dev จริงใช้ + วิธีเขียนย่อให้สะอาด เริ่มฝึกตั้งแต่พื้นฐานจะติดเป็นนิสัย:

1. ตั้งชื่อให้สื่อความหมาย (สำคัญสุด)

❌ อ่านแล้วเดาไม่ออก

let d = 5;
let arr2 = [...];
.box1 { } .box2 { }

✅ ชื่อบอกหน้าที่

let dayCount = 5;
let activeUsers = [...];
.product-card { } .nav-link { }

2. CSS shorthand — สั้นและอ่านง่ายกว่า

❌ ยาวเกินจำเป็น

margin-top: 10px;
margin-right: 20px;
margin-bottom: 10px;
margin-left: 20px;
background-color: #fff;
background-image: url(x.jpg);

✅ รวบ

margin: 10px 20px;

background: #fff url(x.jpg);

3. JS shorthand พื้นฐาน

❌ verbose

let msg = "Hi " + name + "!";

if (age >= 18) { t = "ผู้ใหญ่"; }
else { t = "เด็ก"; }

const double = (n) => { return n * 2; };

✅ ย่อแบบ ES6

let msg = `Hi ${name}!`;       // template literal

const t = age >= 18 ? "ผู้ใหญ่" : "เด็ก";  // ternary

const double = n => n * 2;     // arrow ย่อ
กฎทอง: เขียนย่อ "เพื่ออ่านง่ายขึ้น" ไม่ใช่ "เพื่อโชว์เท่" — ถ้าย่อแล้วคนอ่านงง ให้เขียนยาวกว่าแต่ชัด Clean code = คนอื่น (และตัวเราในอีก 6 เดือน) อ่านแล้วเข้าใจทันที
