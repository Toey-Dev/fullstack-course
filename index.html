🎓 Senior Secure Full Stack

คอร์สเรียน Full Stack + Security แบบ mission-based · TypeScript · Node.js · Next.js · Docker · CI/CD · OWASP

วิธีเรียน: แต่ละสัปดาห์มีบทเรียนเรียงลำดับ อ่าน Mini-lecture → ทำ Mission ด้วยตัวเอง (ห้ามเปิดเฉลยก่อนพยายาม 20 นาที) → ตอบ Quiz ท้ายบท ทุกบทมีตัวอย่างใช้งานจริง เหตุผลเบื้องหลังโค้ดทุกบรรทัด และหลักคิดของ dev

📚 บทเรียน

Week 00 — Frontend Foundation: HTML · CSS · Bootstrap · JS · DOM (ทบทวน) พร้อมเรียน

สรุปแก่นพื้นฐาน + หลักคิด dev — สำหรับทบทวนเร็ว หรือคนใหม่เริ่มจากศูนย์ (อ้างอิงคอร์ส 021.wwry.net) · ~3-4 ชม.

Week 01 — ปิดจบ Frontend: Async/Await · Fetch · LocalStorage · Reading List พร้อมเรียน

Mini-lecture 2 บท + Mission ใหญ่ 1 ชิ้น (แอป My Reading List + deploy จริง) · ~10 ชม.

Week 02 — Node.js + Express: กำเนิด library-system พร้อมเรียน

จาก browser สู่ server · Express API แรก · Books CRUD + validation + ขึ้น GitHub · ~10 ชม.

Week 03 — Express Core + เริ่มเขียน Test (Vitest/Supertest) พร้อมเรียน

Error handling + 404 · ทำไมต้อง test + Test Pyramid · Vitest unit test · Supertest API test + coverage · ~10 ชม.

Week 04 — Docker + PostgreSQL + Zod Validation พร้อมเรียน

Docker (container/compose) · PostgreSQL + SQL · กัน SQL Injection · Zod validation · library-system ใช้ DB จริง · ~10 ชม.

Week 05 — JWT Authentication + Protect Routes พร้อมเรียน

Auth vs AuthZ · bcrypt · JWT ถึงแก่น · auth middleware + role · transaction ยืม-คืน · test auth · ~10 ชม.

Week 06 — TypeScript I: แปลง backend เป็น TS พร้อมเรียน

ทำไม TS · types/interface/union · generics · utility types · z.infer (เชื่อม Zod) · strict mode · ~10 ชม.

Week 07 — TypeScript II: Patterns ที่ทำให้โค้ดแข็งแรง พร้อมเรียน

narrowing + type guards · discriminated unions (make impossible states impossible) · generic constraints · generic repository · ~10 ชม.

Week 08 — Next.js + Tailwind + Full-Stack (จบเฟส 1) พร้อมเรียน

Next.js App Router · server/client component · Tailwind · เชื่อม API + 4 สถานะ · login + token ปลอดภัย · full-stack app · ~10 ชม.

Week 09 — Auth Pages + Protected Routes + App Shell (เฟส 2) พร้อมเรียน

register/login pages · middleware.ts protect (กัน 3 ชั้น) · route groups + layout · sidebar/topbar + user info · ~10 ชม.

Week 10 — CRUD Pages + Dashboard (ปิด Day 8) พร้อมเรียน

controlled form (add/edit เดียว) · delete + refetch · dashboard (aggregate + stat cards) · borrow/return UI · ~10 ชม.

Week 11 — Dockerize App + Compose (app+db+redis) พร้อมเรียน

Dockerfile multi-stage · layer caching · compose 3 service · Redis caching + invalidation · clone แล้วรันได้ · ~10 ชม.

Week 12 — Docker Production + Security Hardening พร้อมเรียน

non-root + minimal image · image scanning (CVE) · secret management · prod compose (healthcheck/limit/restart) · dockerize frontend · ~10 ชม.

Week 13 — CI: GitHub Actions (lint→test→build) พร้อมเรียน

CI concept · workflow/job/step · pipeline + PostgreSQL service · caching · secrets · branch protection + badge · ~10 ชม.

Week 14 — CD: Build Image + Deploy (Registry + SSH) พร้อมเรียน

CD pipeline · build & push GHCR · tag SHA · deploy SSH · rollback · push→deploy อัตโนมัติ · ~10 ชม.

Week 15 — Observability: Logging (pino) + Sentry (จบเฟส 2) พร้อมเรียน

structured logging + correlation ID · Sentry error tracking · staging env · monitoring · log security (ไม่รั่ว PII) · ~10 ชม.

Week 16 — Automation Report เป็น TS Service (โปรเจกต์เสริม · จบเฟส 2) พร้อมเรียน

integration service · typed API client (adapter) · retry/backoff/timeout · scheduled job · allSettled · export · วัดผล impact · ~10 ชม.

Week 17 — OWASP Top 10 ฝั่งป้องกัน + Hardening (เฟส 3) พร้อมเรียน

OWASP Top 10 (defensive) · helmet · rate limiting · CSP · CORS · SSRF · defense report · ~10 ชม.

Week 18 — OAuth 2.0 / OIDC + Token Storage (เฟส 3) พร้อมเรียน

OAuth flow + PKCE · OIDC (id_token) · access/refresh + rotation · token storage (ปิดประเด็น W1) · ช่องโหว่ OAuth · ~10 ชม.

Week 19 — Security ใน CI (DevSecOps) (เฟส 3) พร้อมเรียน

shift left · npm audit/Dependabot · gitleaks (secret) · Semgrep SAST · security gate ใน CI · ~10 ชม.

Week 20 — Pentest โปรเจกต์ตัวเอง → รายงาน attack→fix (จบเฟส 3) พร้อมเรียน

pentest methodology · เครื่องมือ (Burp/ZAP) · เจาะ library-system เอง · pentest report มืออาชีพ · attack→fix→verify · ~10 ชม.

Week 21 — System Design: Performance & Scalability (เฟส 4) พร้อมเรียน

indexing + EXPLAIN · N+1 problem · pagination (offset vs cursor) · caching + invalidation · queue (BullMQ) · ~10 ชม.

Week 22 — Capstone: Design + Kickoff (เฟส 4) พร้อมเรียน

เลือกโปรเจกต์ (VulnTracker/SecureShare) · design doc + threat model (STRIDE) · MVP scoping · scaffold + CI · vertical slice แรก + deploy · ~10 ชม.

Week 23 — Capstone: Complete + Harden (เฟส 4) พร้อมเรียน

should-have features + DoD · test 3 มิติ (happy/edge/error) · self-pentest OWASP · perf pass (W21) · case study + tag v1.0 · ~10 ชม.

Week 24 — Resume · Portfolio · Mock Interview (ปิดคอร์ส) 🎓 พร้อมเรียน · สัปดาห์สุดท้าย

resume XYZ ตรงฝีมือ · portfolio/GitHub/LinkedIn · STAR + system design · เจรจาเงินเดือน · แผนกันลืม (spaced review) · ~10 ชม.
