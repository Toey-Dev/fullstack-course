🧹 Clean Code ประจำ Week 11

1. Multi-stage — แยก build จาก run (image เล็ก)

❌ stage เดียว (image ใหญ่)

FROM node:20
COPY . .
RUN npm install && npm run build
CMD ["node", "dist/server.js"]
# image มี dev deps + source + build tools = ใหญ่

✅ multi-stage (เอาแค่ที่ต้องรัน)

FROM node:20 AS build
RUN npm ci && npm run build
FROM node:20-slim AS run
COPY --from=build /app/dist ./dist
# image มีแค่ dist + prod deps = เล็กกว่ามาก

2. .dockerignore — อย่าก๊อป node_modules/secret เข้า image

node_modules, .env, .git, dist → ใส่ .dockerignore (เหมือน .gitignore) — image เล็กลง + ไม่หลุด secret เข้า image

กฎทอง: multi-stage เสมอ (เล็ก+ปลอดภัย) · .dockerignore · ใช้ node:slim/alpine สำหรับ run · ไม่ hardcode secret ใน Dockerfile (ใช้ env) · pin เวอร์ชัน image (node:20 ไม่ใช่ node:latest)
