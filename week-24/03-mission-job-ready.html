ปิดคอร์ส 24 สัปดาห์ — พร้อมสมัครงานจริง + แผนไม่ให้ลืม

🎯 เป้าหมาย Mission สุดท้าย

จบ Mission นี้ = คุณมีชุดสมัครงานครบ (resume + portfolio + LinkedIn), ซ้อมสัมภาษณ์จนพูดลื่น, เตรียมเจรจาเงินเดือน, และมีแผนกันลืม 6 เดือน · นี่คือคำตอบครบทั้ง 3 โจทย์ที่คุณตั้งไว้วันแรก

🪜 4 Mission ย่อย

Mission 24.1 — Resume + Portfolio + LinkedIn 2 ชม.

  1. เขียน resume 1 หน้า (bullet XYZ, จุดขาย pentester ขึ้นหัว) → export PDF
  2. Pin 3 repo + เขียน Profile README + อัปเดต vee-securedev.vercel.app
  3. LinkedIn: headline + about + projects (link capstone) ตรงกับ resume
✅ เช็ก: ทุก bullet มี link/ตัวเลข · repo pin clone-รันได้ · resume/portfolio/LinkedIn เล่าเรื่องเดียวกัน

Mission 24.2 — ซ้อมเล่า Capstone + Behavioral 90 นาที

  1. เขียนสคริปต์เล่า capstone แบบ STAR (เวอร์ชัน 2 นาที + 5 นาที)
  2. เตรียมตอบ follow-up (ทำไมเลือก X, IDOR แก้ยังไง, scale 100x, จะทำต่างยังไง)
  3. เตรียม behavioral 5 ข้อ (เปลี่ยนสายทำไม, จุดแข็ง/อ่อน, conflict, ล้มเหลว) — อิงเรื่องจริง
เช็ก: อัดเสียงเล่า capstone → ฟังเอง → ปรับจุดวกวน · ตอบ follow-up ได้ลึกทุกข้อ

Mission 24.3 — AI Mock Interview เต็มรูปแบบ 90 นาที

  1. ให้ AI เป็น interviewer: technical + system design (EN) + project deep-dive + behavioral
  2. ตอบสด คิดออกเสียง · รับ feedback → จดจุดอ่อน notes/interview-prep/
  3. ทำซ้ำ system design 1 ข้อจนเล่าได้ลื่น (ใช้กรอบ 5 ขั้น)
✅ เช็ก: ผ่าน mock ครบ 4 ส่วน · มีลิสต์จุดอ่อนที่ต้องฝึกต่อ · system design ตอบ EN ได้

Mission 24.4 — แผนกันลืม + เริ่มสมัครงาน 2 ชม.

  1. ทำ SPACED-REVIEW.md — ตารางทบทวน (โจทย์ที่ 3: เรียนแล้วลืม)
  2. ลิสต์บริษัทเป้าหมาย 10-15 ที่ (ตาม career-paths) + ปรับ resume ต่อ JD
  3. ส่งสมัครชุดแรก 3-5 ที่ · ตั้ง routine: code ทุกวัน + 1 system design/สัปดาห์ + CTF/bug bounty
✅ เช็ก: มีแผน review เป็นรูปธรรม · สมัครจริงแล้ว · มี routine กันสนิม

🧠 แผนกันลืม (โจทย์ที่ 3 ของคุณ)

"เรียนแล้วลืมเมื่อไม่ได้ใช้" แก้ได้ด้วย spaced repetition + active recall — ทบทวนก่อนจะลืมพอดี และดึงความรู้ออกมาเอง (ไม่ใช่อ่านซ้ำ):

ความถี่ทำอะไร
ทุกวัน (15 นาที)code อย่างน้อยนิดเดียว · 1 CTF/bug bounty challenge หรือ refactor โค้ดเก่า
ทุกสัปดาห์1 system design (เขียน+เล่าออกเสียง) · review quiz ของ 1 สัปดาห์เก่า (สุ่ม)
ทุกเดือนเพิ่ม 1 feature ลง capstone (ใช้ของเก่า+ของใหม่) · เขียน blog 1 โพสต์จากสิ่งที่เรียน
ทุก 3 เดือนrebuild mini-project จากศูนย์ (ไม่ดูโค้ดเก่า) · วัดว่าจำได้แค่ไหน → review จุดที่ลืม
💡 หัวใจ: ความรู้ที่ใช้ไม่ลืม · เปลี่ยนจาก "เรียนเพื่อรู้" เป็น "ใช้เพื่อจำ" · capstone + งานจริงที่ media-presto + CTF = สนามใช้ความรู้ต่อเนื่อง · การ "สอน/เขียน blog" สิ่งที่เรียน = active recall ที่ทรงพลังสุด

🛤️ เส้นทางต่อ — สู่ AI-Security Engineer

คอร์สนี้พาคุณถึง Secure Full Stack · ก้าวต่อไปตาม FIVE-YEAR-PLAN:

  • ระยะสั้น (0-1 ปี): ได้งาน secure full stack 60-80k · สะสม impact ที่วัดผลได้ · ต่อยอด CTF/bug bounty
  • ระยะกลาง (1-3 ปี): ขยับ senior / ลึกสาย security · เริ่มจับ AI/ML security (prompt injection, model security, LLM app sec)
  • ระยะยาว (3-5 ปี): AI-Security Engineer — รวม full stack + offensive security + AI · ตำแหน่งหายากและจ่ายสูง
คุณพร้อมแล้ว: มีฐาน full stack แข็ง + พื้นเพ pentester + ระบบกันลืม + portfolio จริง · เหลือแค่ลงสนาม สิ่งที่เคยเป็นจุดอ่อน (ไม่มี cert แพง) คุณแก้ด้วย proof จริง (capstone + self-pentest + free certs) ซึ่งทรงพลังกว่า
