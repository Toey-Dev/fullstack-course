🧹 Clean Code ประจำ Week 4

1. Schema เป็น single source of truth ของ "ข้อมูลที่ถูกต้อง"

❌ เช็คมือ กระจาย ลืมง่าย

if (!title) return res.status(400)...
if (typeof title !== "string") ...
if (title.length > 200) ...
if (price < 0) ...  // ซ้ำทุก route

✅ Zod schema ที่เดียว ใช้ซ้ำ

const BookSchema = z.object({
  title: z.string().min(1).max(200),
  price: z.number().nonnegative(),
});
// ใช้ได้ทุก route + แปลง type ให้ + error สวย

2. อย่าต่อ SQL ด้วย string (SQL Injection)

❌ ช่องโหว่ร้ายแรง

db.query(`SELECT * FROM books
  WHERE id = ${req.params.id}`);
// id = "1; DROP TABLE books" = หายนะ

✅ parameterized query

db.query("SELECT * FROM books WHERE id = $1",
  [req.params.id]);
// ค่าถูก escape อัตโนมัติ ปลอดภัย
กฎทอง: validate ทุก input ด้วย schema · ไม่ต่อ SQL ด้วย string เด็ดขาด · เก็บ config (DB password) ใน .env ไม่ commit · แยก repository layer (โค้ดคุย DB) ออกจาก controller
