🧹 Clean Code ประจำ Week 12

1. รัน container ด้วย non-root user

❌ default = root (อันตราย)

FROM node:20-slim
COPY --from=build /app/dist ./dist
CMD ["node", "dist/server.js"]
# รันด้วย root — โดนเจาะ = คุม container ได้เต็ม

✅ non-root user

FROM node:20-slim
COPY --from=build /app/dist ./dist
USER node                  # ⭐ รันด้วย user ธรรมดา
CMD ["node", "dist/server.js"]
# โดนเจาะ = สิทธิ์จำกัด

2. Secret runtime ไม่ใช่ build-time

DB password, JWT secret = ใส่ตอน runtime (env/secret) ไม่ใช่ ARG/ENV ใน Dockerfile (ติดอยู่ใน image layer ถาวร — ใครได้ image ขุดเจอ)

กฎทอง: non-root เสมอ · minimal image (slim/distroless) · scan ก่อน deploy · secret runtime ไม่ฝัง image · pin version · resource limit กัน 1 service กิน CPU/RAM หมด
